## Quick Reference

### Common Tasks

- **Capture an idea**: Press hotkey (default: `Cmd/Ctrl + I`) or click lightbulb icon
- **Process an idea**: Use "Ideate" button or "Classify Current Note" command
- **Validate an idea**: Use "Quick Validate" command
- **Transform an idea**: Use transformation commands (variants, scaffolds, etc.)
- **View all ideas**: Use "Open Dashboard" command
- **See relationships**: Use "Open Graph View" command

### Keyboard Shortcuts

- **Capture**: `Cmd/Ctrl + I` (default)
- **Save**: `Alt/Option + Enter` (in capture modal)
- **Ideate**: `Cmd/Ctrl + Enter` (in capture modal)
- **Command Palette**: `Cmd/Ctrl + P`

### Getting Help

- Look for **(?)** help icons throughout Ideatr for contextual help
- Use **"Open Tutorials"** command to access all tutorials
- Check **Settings → Ideatr** for configuration options
