# Dashboard Guide

The Dashboard provides a comprehensive view of all your ideas with powerful filtering, sorting, and organization tools.

## Understanding the Dashboard

The Dashboard is your central hub for managing ideas. It includes:
- **Table view** - All ideas in a sortable, filterable table
- **Filter panel** - Search and filter by multiple criteria
- **Side panels** - Clusters, resurfacing, and triage inbox
- **Pagination** - Navigate through large idea collections

## Opening the Dashboard

**How to open:**
1. Use **"Open Dashboard"** command (Command Palette → "Ideatr: Open Dashboard")
2. Or use the ribbon icon (if configured)

## Table View

### Columns

The table shows:
- **Title** - Idea title/subject
- **Category** - AI-classified category
- **Tags** - Associated tags
- **Status** - Lifecycle status
- **Created** - Creation date
- **Actions** - Quick actions (open, edit, etc.)

### Sorting

**How to sort:**
- Click column headers to sort
- Click again to reverse sort order
- Sort by any column (title, category, status, date)

### Pagination

**How to paginate:**
- Use pagination controls at bottom
- Configure items per page in settings
- Default: 50 items per page

## Filtering

### Search

**Text search:**
- Search across idea content, titles, and metadata
- Real-time filtering as you type
- Searches all fields

### Category Filter

**Filter by category:**
- Enter comma-separated categories
- Example: `SaaS,Productivity,Tool`
- Shows ideas matching any of the categories

### Tags Filter

**Filter by tags:**
- Enter comma-separated tags
- Example: `automation,ai,productivity`
- Shows ideas with any of the tags

### Status Filter

**Filter by status:**
- Select from dropdown
- Options: All, Captured, Validated, Promoted, Archived
- Shows ideas with selected status

### Date Range Filter

**Filter by date:**
- Enter date range (e.g., `2025-01-01 to 2025-01-31`)
- Filter by creation date
- Useful for finding recent or old ideas

### Clearing Filters

**How to clear:**
- Click "Clear Filters" button
- Or manually clear each filter field
- Filters persist across sessions (if enabled in settings)

## Side Panels

### Clusters Panel

**What it shows:**
- Mini-graph of idea clusters
- Visual representation of relationships
- Click clusters to filter ideas

**How to use:**
- View related ideas grouped together
- Click cluster to see ideas in that cluster
- Understand idea relationships at a glance

### Resurfacing Panel

**What it shows:**
- Old ideas that haven't been touched recently
- Ideas older than threshold (default: 7 days)
- Helps prevent forgetting ideas

**How to use:**
- Review old ideas regularly
- Click to open and review
- Decide whether to reactivate or archive

### Triage Inbox

**What it shows:**
- Unclassified ideas (no category or tags)
- Ideas that need attention
- Ideas that haven't been processed

**How to use:**
- Review unclassified ideas
- Use "Classify Current Note" to classify
- Process ideas in bulk

## Dashboard Workflow

### Daily Workflow

1. **Open Dashboard** - Start your session
2. **Check Triage Inbox** - Review unclassified ideas
3. **Review Resurfacing** - Check old ideas
4. **Filter by Status** - Focus on specific stages
5. **Process Ideas** - Classify, validate, transform

### Weekly Workflow

1. **Review Clusters** - See idea relationships
2. **Filter by Date** - Review recent ideas
3. **Archive Old Ideas** - Clean up vault
4. **Promote Validated Ideas** - Move forward

## Settings

Configure Dashboard in **Settings → Ideatr → Dashboard Settings**:
- **Items per page** - Number of ideas per page
- **Default view** - Table, graph, or inbox
- **Auto-refresh** - Refresh automatically
- **Persist filters** - Save filter state across sessions

## Tips

- **Use filters effectively** - Combine multiple filters for precise results
- **Sort regularly** - Sort by date to see newest/oldest ideas
- **Check triage inbox** - Process unclassified ideas regularly
- **Review clusters** - Understand idea relationships
- **Resurface old ideas** - Don't forget about older ideas
